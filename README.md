# PulseDL

Electron frontend for `yt-dlp` using Node.js and TypeScript.

## Stack

- Electron
- Node.js
- TypeScript (main + preload)
- Vanilla HTML/CSS/JS renderer

## Run

1. Install dependencies:
   `npm install`
2. Start the app:
   `npm start`

## Notes

- Downloads are written to your system Downloads folder.
- `yt-dlp` must be installed and available in your PATH.
